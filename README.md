## SDCS Weather App

Check the weather in any city you like around the world. 

## Technologies used
* React
* react-redux 
* bootstrap
* webpack
* axios
* chai
* nightmare
* Express

## Setup
Clone/download the repo onto your local system then run the following commands:
```
npm install
```
```
npm run build
```
```
npm start
```
Then go to http://localhost:3000 in your browser

## To test
```
npm run build
```
```
npm test
```
